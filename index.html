¿Qué es el miércoles de web?

Cada segundo miércoles del mes, se realiza una tertulia sobre tecnologías web en co-labora, la oficina coworking en Managua. Se brinda un espacio agradable y creativo con conferencias interesantes de las últimas tecnologías web. Además tienen la posibilidad de conectarse con otros profesionales de esta área, conversar e intercambiar experiencias e ideas innovadoras.

Trabajar CSS con estilo

Las hojas de estilo css contienen toda la información necesaria para definir el diseño de un sitio web y los proyectos web en la actualidad se caracterizan por el involucramiento de varias personas en el desarrollo y por tener una estructura de código organizada que permita un fácil mantenimiento y escalabilidad. Esta charla tiene el objetivo de introducirles al uso de preprocesadores de CSS, una mejor forma de trabajar código CSS dinámico, en donde realmente se puedan aprovechar el uso de variables, funciones y pruebas lógicas dentro de nuestras definiciones de CSS.

Mapas que cuentan historias

¿Quiere crear un mapa con un diseño distinto? ¿Quiere mostrar datos según su caso de uso? En vez de usar los mapas limitantes de Google (u otras fuentes cerradas) se presetará las herramientas Mapbox y OpenStreetMap, las cuales le permiten a los constuctores de sitios web, diseñar mapas individualmente y mostrar la información requerida de la manera deseada.

Web Project Management

Profesionales independientes usualmente se enfocan en su vocación, ya sea diseño o programación; pero gran parte del trabajo diario es la administración de proyectos. Siendo artistas o ninjas del código no significa que se debe descuidar los aspectos de negocio relacionados al diseño y desarrollo web. El emprendedor Harold Maduro compartirá experiencias y contará sobre herramientas de manejo de proyectos web.
